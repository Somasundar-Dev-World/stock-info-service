About StockIQ

StockIQ is a real-time market insights dashboard built with Angular 17 and Spring Boot. It leverages the Alpha Vantage API to deliver up-to-the-minute stock quotes, company information, and the latest financial news with sentiment analysis.

Real-time Data

Get the latest prices, volume, and market cap for your favorite stocks.

News Sentiment

Read the latest news and see whether the market sentiment is Bullish or Bearish.

Responsive Design

Beautiful glassmorphism design that works seamlessly on desktop and mobile.
